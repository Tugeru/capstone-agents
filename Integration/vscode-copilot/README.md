# VS Code Copilot Integration

This folder contains resources for integrating Capstone Agents with VS Code Copilot.

## Setup
Run `python scripts/setup_vscode_copilot.py` to generate the `.vscode/tasks.json` and workspace file.

## Usage
1. Open `capstone-agents.code-workspace`.
2. Open an agent file (e.g., `agents/frontend/frontend-planning.md`).
3. Copy the content.
4. Paste into Copilot Chat.
